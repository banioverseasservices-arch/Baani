# Deployment Guide - BAANI Overseas Services

## 🚀 GitHub Pages Deployment

Your website is ready to be deployed on GitHub Pages! Follow these steps to go live:

### Step 1: Enable GitHub Pages
1. Go to: `https://github.com/banioverseasservices-arch/Baani/settings/pages`
2. Under "Source", select:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
3. Click **Save**
4. Wait 2-3 minutes for deployment

### Step 2: Access Your Live Website
Your website will be available at:
```
https://banioverseasservices-arch.github.io/Baani/
```

## 🔧 Customization Guide

### Update Contact Information:
Edit `index.html` and find the contact section:
```html
<p>+91 XXXXXXXXXX</p>  <!-- Replace with your phone -->
<p>info@baanioverseas.com</p>  <!-- Update email -->
<p>Your Office Address</p>  <!-- Update address -->
```

### Change Color Scheme:
Search for `#d4af37` (gold) and `#000` (black) in the CSS section and update with your preferred colors.

### Update Services:
Modify the services cards section in the HTML to add/edit your services.

### Update FAQs:
Edit the FAQ section with your own questions and answers.
